# Permissions

Allowed in Sprint 1:

- health checks;
- read-only authentication proof;
- read-only metadata requests;
- redacted raw storage;
- mock adapter calls;
- tests and migrations.

Forbidden:

- order creation or modification;
- price, menu, discount, purchasing, payment, or employee changes;
- direct Hermes database access;
- production deployment without an approved runbook;
- secrets in logs, Git, Drive, or Telegram.
